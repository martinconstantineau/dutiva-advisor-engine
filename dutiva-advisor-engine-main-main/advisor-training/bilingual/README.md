# Bilingual Strategy — Dutiva Advisor

## Current State

The Dutiva Advisor MVP operates in **English only**. The guidance pipeline generates a French placeholder field (`advisor_answer_fr_placeholder`) for every guidance card, but these placeholders are **not validated** and must not be presented to users.

---

## Bilingual Pipeline Plan (Phase 2)

### How EN/FR Pairing Works

Each guidance card produced by the pipeline has a 1:1 correspondence with a bilingual record:

```
normalized record (EN source)
  → guidance card (advisor_answer_en + advisor_answer_fr_placeholder)
  → bilingual record (advisor_answer_en + advisor_answer_fr [validated])
```

### French Content Requirements

1. **Do not use raw machine translation for legal content.** Machine translation of legal text produces errors that can mislead employers.
2. French text must be reviewed by a qualified bilingual person with knowledge of Canadian HR/employment law terminology.
3. Use official French statute titles:
   - Canada Labour Code → *Code canadien du travail*
   - Canadian Human Rights Act → *Loi canadienne sur les droits de la personne*
   - Work Place Harassment and Violence Prevention Regulations → *Règlement sur la prévention du harcèlement et de la violence dans le lieu de travail*
4. For Quebec-specific content (Phase 2), use Quebec French legal terminology (e.g., *normes du travail*, *Commission des droits de la personne*).

---

## Bilingual Schema

See `schema.json` in this folder for the complete bilingual guidance card schema.

Key fields:
- `advisor_answer_fr` — validated French answer (not the pipeline placeholder)
- `advisor_answer_fr_status` — must be `"approved"` before being shown to users
- `citation_fr` — French statute citation
- `law_title_fr` — official French law title
- `fr_reviewer` + `fr_reviewed_at` — reviewer accountability

---

## Validation States

| Status | Meaning |
|--------|---------|
| `placeholder` | Auto-generated by pipeline — do NOT show to users |
| `machine_draft` | Initial machine translation — do NOT show to users |
| `reviewed` | Reviewed but not yet formally approved |
| `approved` | Validated by qualified reviewer — safe to show to French users |

---

## French Content Roadmap

| Phase | Content |
|-------|---------|
| MVP | EN only. FR placeholders stored in guidance pipeline — not exposed. |
| Phase 2a | FR translations of high-risk guidance cards (Termination, Harassment, Accommodation) reviewed and approved |
| Phase 2b | Full FR coverage of all guidance cards |
| Phase 3 | FR versions of prompts, response style guide, and system prompt |
| Phase 4 | FR version of test cases and evaluation suite |
